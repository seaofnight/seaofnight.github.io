vagrant-install

// under vagrant virtualbox

개요

  • 로컬 PC vagrant 를 설치
  • vagrant 를 통한 버추얼 박스에 리소스 부여 방법 등을 설명하겠다.

Vagrant 설치

  • 본인 PC 은 Manjaro Linux 라서 크게 어려움이 있지는 않았다.
    • 윈도우키 "software" 입력 후 pacman 화면에서 vagrant 를 입력하여 설치 버튼을 클릭하면 끝난다. 
$ sudo pacman -Sy virtualbox virtualbox-host-modules net-tools
$ sudo vagrant -v 
Vagrant 2.2.5

vagrant 리소스 할당 방법

  • Vagrant 는 공식홈페이지의 메뉴얼에 나온 내용대로 vagrant init 하면 VagrantFile 이 생성이된다.
  • 이 파일에 VM 의 스펙과 리소스 할당 등을 적어서 vagrant up 하면된다.
  • Vagrant 는 Ruby 로 작성이 되어있어서 당연히 Ruby 와 친숙해지면 더 잘 운영할 수 있을것이다.
  • 우선 가지고 있는 소스의 내용을 적고 부분부분 설명을 하겠다. 
[root@di7-03 vagrant]# cat Vagrantfile 
# -*- mode: ruby -*-
# vi: set ft=ruby :

require 'ipaddr'
load './config.rb'
image = "centos/7"

def nodeIP(ips, id)
  return ((IPAddr.new ips)|(2+id)).to_s()
end

Vagrant.configure("2") do |config|
    config.ssh.insert_key = false
    config.vm.synced_folder ".", "/vagrant", owner: "vagrant", group: "vagrant", rsync__auto: true, type: "virtualbox"

    (0..$worker_count).each do |index|
        node_name = "vitess%d" % index

        config.vm.define node_name do |node|
            node.vm.box = image
            node.vm.hostname = node_name

            # multiple eth interface 
            node.vm.network :private_network, ip: nodeIP($cluster_ips, index)
            node.vm.network :private_network, ip: nodeIP($manage_ips, index)
#           node.vm.network :private_network, ip: nodeIP($private_ips, index)

            node.vm.provider :virtualbox do |vb|
                vb.name = node_name
                vb.cpus = 8 
                vb.memory = 8192 
                vb.customize ["modifyvm", :id, "--natdnshostresolver1", "on"]

                # disk save path on barematal 
                disk = "/data/storage/"
                disk << node_name 
                disk << "/sdb.vdi"

                unless File.exist?(disk)
                        vb.customize ['createhd', '--filename', disk, '--variant', 'Fixed', '--size', $disk_size_mb * 1024]
                end
                vb.customize ['storageattach', :id,  '--storagectl', 'IDE', '--port', 1, '--device', 0, '--type', 'hdd', '--medium', disk]
            end

            # provision script 
            node.vm.provision "shell", 
                name: "Base Installation (scripts/base.sh)",
                path: "scripts/base.sh"

        end
    end
end
